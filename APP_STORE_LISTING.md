# App Store Listing Information

## App Name
AffectAtlas

## Subtitle (iOS) / Short Description (Android)
Map emotions to beautiful design systems

## Description

Navigate the landscape of emotions to discover your perfect design system with AffectAtlas - the innovative app that maps affect to aesthetics through Plutchik's Wheel of Emotions.

**KEY FEATURES:**

🎨 **Interactive Emotion Wheel**
Select from 8 primary emotions with intuitive touch controls. Choose your intensity level: intense, moderate, or mild.

🌈 **Advanced Color Science**
Uses OKLCH color space for perceptually uniform gradients. Generate harmonious color palettes that truly reflect emotional states.

📐 **Complete Design Systems**
Get more than just colors:
• Typography settings
• Spacing scales
• Border radius values
• Animation parameters
• Semantic color tokens

📤 **Multi-Format Export**
Export your design system in multiple formats:
• CSS Custom Properties
• JSON tokens
• Swift code for iOS
• Kotlin code for Android
• SCSS variables
• Tailwind configuration

🎯 **Professional Quality**
• WCAG accessibility compliance
• Industry-standard design tokens
• Production-ready code output
• Copy to clipboard functionality

**PERFECT FOR:**
• UI/UX Designers seeking inspiration
• Developers building design systems
• Brand designers exploring emotional connections
• Students learning color theory
• Anyone interested in the psychology of design

**PRIVACY FIRST:**
• No account required
• No data collection
• No ads
• 100% offline functionality
• Open source

**HOW IT WORKS:**
1. Select an emotion from the wheel
2. Choose your intensity level
3. Preview the generated design system
4. Export in your preferred format

Based on Robert Plutchik's psychoevolutionary theory of emotions, this app creates scientifically-grounded design systems that resonate with human psychology.

Whether you're designing a calming meditation app, an exciting gaming interface, or a trustworthy financial platform, AffectAtlas helps you chart the perfect visual language for your emotional intent.
